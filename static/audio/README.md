# Roadmap audio scaffold

The frontend tries static MP3 first, then falls back to browser TTS.

Naming convention:

- `starter_u1_l1_word_1.mp3`
- `starter_u1_l1_dialogue.mp3`
- `starter_u1_l1_speak_1.mp3`
- `flyer_u1_l1_word_1.mp3`
- `flyer_u1_l1_dialogue.mp3`
- `flyer_u1_l1_speak_1.mp3`

Keep files grouped by level:

- `static/audio/starter/`
- `static/audio/flyer/`
